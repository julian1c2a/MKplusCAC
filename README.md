## Development Workflow

```bash
# Initialize lock system (first time only)
bash git-lock.bash init

# Create a new module
bash new-module.bash ModuleName

# Build
make build

# Check for sorry
make sorry

# Show locked files and sorry status
make status

# Regenerate root import file
bash gen-root.bash
```
